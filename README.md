# PingApp
Program stworzony przez Pawła Malisiewicza i Marka Ciebierę.
Pozwala na sprawdzenie połączenia, straconych pakietów jak i opóźnień.

Działanie:
- W okienku **Podaj adres** należy wprowadzić adres który chcemy sprawdzić.
- Po kliknięciu przycisku **Pinguj!** program rozpoczyna pracę.
- po chwili poniżej przycisku powinny wyświetlić się nade określone przez program.
